# Skiza Workflow

## Overview

Staff/admin can prepare ringtone clips for Safaricom Skiza services.

## Workflow

1. Open track
2. Click 'Sync Skiza'
3. Load editor
4. Select clip
5. Preview
6. Normalize
7. Save draft -> prompt for additional license
8. Submit for approval  -> prompt for additional license
9. Export  -> prompt for additional license
10. Sync to provider  -> prompt for additional license

## Statuses

- draft
- pending_review
- approved
- exported
- failed

## Worker Jobs

- clip generation
- normalization
- transcoding
- provider sync
